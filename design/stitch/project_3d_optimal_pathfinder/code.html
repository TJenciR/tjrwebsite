Implementation Notes

The core algorithm utilizes a custom priority queue implementation tailored for 3D coordinate tuples, significantly reducing the overhead associated with standard library queues when handling large volumetric grids.

Heuristic function employs a modified Euclidean distance that applies heavier weighting to vertical traversal, mimicking real-world constraints often found in drone flight path planning where altitude changes consume more energy.

def heuristic(a: tuple, b: tuple) -> float:
    # Modified Euclidean with vertical weight
    dx = abs(a[0] - b[0])
    dy = abs(a[1] - b[1])
    dz = abs(a[2] - b[2]) * 1.5 # Z-axis penalty
    
    return math.sqrt(dx*dx + dy*dy + dz*dz)
